SECTION 01

왜 행원에게 필요한가

저축은행·새마을금고·캐피탈 행원들은 매일 법률적 판단이 필요한 상황에 직면합니다. 하지만 대부분의 행원에게 법률 전문 지식은 없고, 무엇을 해야 하는지조차 모르는 경우가 많습니다. 변호사에게 물어보자니 비용이 부담되고, 인터넷에서 찾자니 정확성을 보장할 수 없습니다.

현장에서 실제로 발생하는 4가지 상황

상황 행원이 하는 말 문제 결과
연체 채권 처리 "채무자가 3개월째 연체 중인데 어떻게 해야 하나요?" 경매인지, 내용증명인지, 지급명령인지 판단 못 함 시효 소멸 / 담보 가치 하락 / 채권 회수 실패
소멸시효 관리 "소멸시효가 언제 만료되는지 모르겠어요" 시효 계산 기준조차 모르고 있음 시효 완성 → 채권 소멸 → 수백만 원 손실
금감원 민원 대응 "금감원에서 민원 답변하라는데 뭘 써야 하나요?" 법적 근거를 모른 채 답변 작성 불완전 답변 → 추가 민원 / 제재 리스크
채무자 사망 처리 "채무자가 사망했는데 어떻게 채권을 회수하나요?" 상속 절차 및 관련 서류를 모름 처리 지연 → 상속 재산 소멸 → 채권 회수 불가
핵심 문제
행원이 "무엇을 해야 하는지"를 모른다. 법률 지식이 없어서가 아니라, 현재 상황이 어떤 법률적 케이스에 해당하는지 판단할 수 없기 때문이다.

시장 규모

PRIMARY MARKET

새마을금고 3,242개 지점

전국 지점마다 법률 문서 처리 인력 부재. 지점당 연간 수십 건의 법률 서류 발생.

PRIMARY MARKET

저축은행 79개사

법무팀이 없는 중소 저축은행은 외부 법무사·변호사에 의존. 건당 처리 비용 과다.

SECONDARY MARKET

캐피탈 50+ 개사

할부·리스 연체 채권 처리 수요 급증. 인력 확충 없이 처리량 증가 요구.

EXPANSION

신협·농협·수협 지점

동일한 구조적 문제 보유. 뱅크법률집사 검증 후 확장 가능한 인접 시장.

SECTION 02

서비스 개요 — 상황 상담 → 자동 서류 안내

핵심 컨셉
행원이 상황만 설명하면 AI가 법률 케이스를 자동 판별하고, 필요한 서류와 절차를 즉시 안내합니다. 행원은 법률 지식 없이도 정확한 서류를 생성할 수 있습니다.

4가지 핵심 기능

① 상황 상담

채팅 또는 구조화 폼으로 상황 입력

행원이 자연어로 상황을 설명하거나, AI가 제시하는 질문에 답합니다. 채팅 형태로 3~5회 질문 응답만으로 케이스 판별 완료.

② 자동 케이스 판별

AI가 법률 케이스 + 필요 서류 안내

의사결정 트리 + RAG 기반으로 10가지 법률 케이스 중 해당 케이스를 자동 판별. 필요 서류 목록과 절차를 즉시 안내.

③ AI 초안 생성

정보 입력 → 완성된 서류 초안 즉시 생성

채무자 정보, 금액, 날짜 등 기본 정보만 입력하면 AI가 법률 양식에 맞는 서류 초안을 자동 생성. 평균 소요 시간 5~15분.

④ 변호사 검토 연계

AI 초안 → 변호사 최종 검토 → 제출

고위험·고금액 건은 법무법인 더 에이치 황해 변호사가 직접 검토. 셀프 다운로드 또는 검토 요청 선택 가능.

SECTION 03

행원이 만나는 10가지 실무 상황

행원이 실제로 맞닥뜨리는 상황 10가지를 "행원이 하는 말" 중심으로 정리했습니다. AI는 이 발화를 분석해 법률 케이스를 판별하고 필요한 서류를 안내합니다.

01 "채무자가 연체 3개월인데 담보가 있어요" 서류 7건
AI 판단 결과
임의경매 신청

부동산 담보 있음 + 3개월+ 연체 → 담보권 실행 (임의경매) 절차 개시 필요

필요 서류 목록
  • 임의경매신청서
  • 근저당권설정계약서 사본
  • 채권최고액 확인서
  • 대출약정서 사본
  • 채권계산서
  • 부동산 등기사항전부증명서
  • 내용증명 (최고서)
02 "이 채권 시효가 곧 끝나는 것 같아요" 서류 3건
AI 판단 결과
소멸시효 중단 내용증명

시효 만료 임박 → 내용증명 발송으로 시효 중단 (민법 제174조)

필요 서류 목록
  • 소멸시효 중단 내용증명
  • 채권 원장 (시효 기산점 확인용)
  • 발송 확인서 (내용증명 우체국 영수증)
03 "채무자한테 돈 받아내고 싶어요" 서류 4건
AI 판단 결과
지급명령 신청

무담보 연체 채권 → 법원 지급명령 신청 (민사소송법 제462조)

필요 서류 목록
  • 지급명령신청서
  • 채권계산서 (원금·이자·지연손해금)
  • 대출약정서 사본
  • 채무자 주민등록초본
04 "판결문은 있는데 돈을 안 줘요" 서류 3건
AI 판단 결과
채권 압류 및 전부명령

집행권원(판결) 보유 → 급여·예금 등 채권 압류 및 전부명령 신청

필요 서류 목록
  • 채권압류 및 전부명령 신청서
  • 판결문 정본 (집행문 포함)
  • 채무자 재산 조회서 (금융거래정보 등)
05 "금감원에서 민원 답변하래요" 서류 3건
AI 판단 결과
금감원 민원 답변서

금융감독원 민원 접수 → 법적 근거 기반 공식 답변서 작성 필요

필요 서류 목록
  • 금감원 민원 답변서
  • 관련 거래 내역서
  • 관련 약관·규정 발췌본
06 "새 대출에 근저당 설정해야 해요" 서류 4건
AI 판단 결과
근저당권 설정 등기

신규 담보 대출 → 근저당권 설정 등기 신청 및 관련 서류 준비

필요 서류 목록
  • 근저당권설정계약서
  • 부동산 등기사항전부증명서
  • 인감증명서 (채무자)
  • 등기 신청서
07 "채무자가 사망했어요" 서류 3건
AI 판단 결과
상속 채권 처리

채무자 사망 → 상속인 확인 후 상속 재산에 대한 채권 행사 절차

필요 서류 목록
  • 상속인 확인 통보서
  • 가족관계증명서 (채무자)
  • 상속재산 한정승인·포기 확인서
08 "연체 등록 전에 통지해야 하나요?" 서류 2건
AI 판단 결과
신용정보원 사전통지

연체 등록 전 의무적 사전 통지 (신용정보법 제25조) 이행 필요

필요 서류 목록
  • 연체 등록 사전통지 공문
  • 발송 확인서 (등기우편 영수증)
09 "고객이 대출 철회하겠대요" 서류 2건
AI 판단 결과
청약 철회 처리

대출 청약 철회권 행사 (대부업법 제5조) → 14일 이내 전액 반환 처리

필요 서류 목록
  • 청약 철회 신청서
  • 반환 처리 확인서
10 "부실채권 매각 서류가 필요해요" 서류 3건
AI 판단 결과
NPL 매각

부실채권(NPL) 매각 → 채권 양도 계약 및 채무자 통지 절차 필요

필요 서류 목록
  • 채권 양도·양수 계약서
  • 채권 양도 통지서 (채무자 발송용)
  • 채권 원장 및 잔액 확인서

SECTION 05

기술 구조 — 아파트집사 엔진 전환

뱅크법률집사는 아파트집사에서 검증된 AI 문서 처리 엔진을 기반으로 구축합니다. 기존 모듈 재활용으로 개발 비용·기간을 최소화하고, 법률 특화 모듈을 추가합니다.

모듈 구성

재활용

문서 생성 엔진

아파트집사의 템플릿 기반 문서 자동 생성 모듈. 필드 매핑·PDF 변환 검증 완료.

재활용

사용자 인증 / 권한 관리

기관별 행원 계정 관리, 역할 기반 접근 제어(RBAC) 모듈 그대로 적용.

재활용

알림 / 스케줄러

시효 만료 D-day 알림, 검토 마감 알림 등 기존 스케줄러 재활용.

신규

상황 판단 AI 엔진

의사결정 트리 + RAG(검색증강생성) 기반. 행원 발화 → 10개 법률 케이스 자동 판별. 핵심 개발 항목.

신규

채팅 상담 인터페이스

자연어 입력 + 구조화 질문 응답. AI 판단 결과 카드 UI. WebSocket 기반 실시간 처리.

신규

법률 서류 템플릿 DB

10개 케이스 × 평균 3.4건 = 34개 법률 서류 양식. 법원별·기관별 최신 양식 유지 관리.

구분 항목 기간 비고
Phase 1 상황 판단 AI 엔진 개발 + 채팅 UI 4주 핵심 기능
Phase 2 10개 케이스 서류 템플릿 구축 + 연동 3주 법률 검증 포함
Phase 3 변호사 검토 워크플로우 + 파일럿 2~3주 고객사 1곳 파일럿
합계 9~10주 총 개발 8~12주 범위 내

법률N

법률 정보 포털

정보 검색 위주. 서류 생성 불가. 상황 판단 AI 없음. 행원이 스스로 케이스를 찾아야 함.

뱅크솔루트

금융권 업무 자동화

여신·수신 업무 자동화 중심. 법률 서류 특화 없음. 케이스 판별 AI 없음.

뱅크법률집사 (당사)

금융권 행원 전용 — 상황 상담 → 케이스 판별 → 서류 생성 원스톱

행원이 상황만 말하면 AI가 법률 케이스를 자동 판별하고 서류를 즉시 생성. 경쟁 서비스 중 유일한 "상황 → 판단 → 서류" 원스톱 플로우. 변호사 검토 연계로 책임 소재 명확화.
