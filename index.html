01

왜 행원에게 필요한가

저축은행·새마을금고 행원의 하루

저축은행과 새마을금고의 행원은 대출 실행, 연체 관리, 채권 회수, 금감원 민원 대응까지 한 사람이 모두 처리해야 한다. 그러나 전담 법무팀은 없다. 법률 문서가 필요한 순간마다 변호사를 찾거나, 인터넷 검색으로 양식을 구하거나, 선배에게 물어보는 구조다. 이 비효율이 기관 전체의 리스크로 이어진다.

핵심 문제
전담 법무팀 없는 상황에서 행원 개인이 경매 신청서부터 금감원 민원 답변서까지 처리해야 한다. 서류 오류 한 건이 소멸시효 도과, 경매 각하, 민원 악화로 직결된다.

4가지 고통 시나리오

CASE 01

경매 서류를 모른다

담보 부동산 임의경매를 신청해야 하는데, 필요한 서류가 7가지인지 3가지인지조차 모른다. 제출 누락 시 경매 신청이 각하된다.

CASE 02

소멸시효 만료 위기

채권 소멸시효가 D-7임을 아무도 몰랐다. 내용증명 발송 기한을 놓치면 채권 자체가 소멸되고 기관 손실로 직결된다.

CASE 03

금감원 민원 답변서

고객이 금감원에 민원을 제기했다. 답변 기한은 10영업일. 적합한 형식과 내용을 모르면 민원 악화 및 2차 제재 리스크가 발생한다.

CASE 04

불완전판매 민원

대출 상품 불완전판매 주장이 접수됐다. 어떤 서류를 준비하고, 어떤 사실관계를 기술해야 하는지 행원은 알 수 없다.

시장 규모

업종 기관 수 행원 수 (추정) 법률 서류 니즈
새마을금고 3,242개 약 70,000명 경매, 채권 회수, 민원 대응
저축은행 79개 약 15,000명 경매, NPL, 금감원 대응
캐피탈사 50개+ 약 20,000명 채권 압류, 소멸시효, NPL
대부업체 8,000개+ 약 30,000명 지급명령, 압류, 내용증명
시장 기회
새마을금고 3,242개 중 독립 점포 기준으로도 월 구독 모델 적용 시 수십억 원 시장. 저축은행·캐피탈·대부업까지 확장하면 국내 소형 금융기관 전체가 잠재 고객이다.
02

서비스 개요 — 로폼 for 금융권

뱅크법률집사는 저축은행·새마을금고·캐피탈 등 소형 금융기관 행원을 위한 금융권 특화 법률 서류 자동화 플랫폼이다. 로폼(LawForm)의 개념을 금융 실무에 맞게 재설계했다. 케이스 선택 → 서류 목록 안내 → AI 초안 생성 → 변호사 검토 연계까지 원스톱으로 처리한다.

핵심 4가지 기능

📋

① 케이스별 서류 안내

케이스 선택 → 필요 서류 목록 + 체크리스트 + 제출기한 자동 제공. 무엇을 준비해야 하는지 1분 안에 파악.

✍️

② 서류 작성 가이드 + AI 초안

필드별 작성 가이드 + 계좌번호·채무자 정보 입력 시 AI가 초안 자동 생성. 작성 오류를 사전에 차단.

⚖️

③ 변호사 검토 연계

클릭 한 번으로 법무법인 황해에 검토 요청. SLA 24~48시간 내 답변. 법무법인 직결로 신뢰성 확보.

📚

④ 레퍼런스 축적

기관 전용 라이브러리. 담당자가 교체돼도 과거 서류와 처리 이력이 유지됨. 기관 법무 노하우의 자산화.

05

아파트집사 엔진 전환 기술 구조

뱅크법률집사는 기존 아파트집사 플랫폼의 핵심 엔진을 재활용하여 개발 기간과 비용을 최소화한다. 새로 만들어야 하는 것만 만든다.

그대로 활용하는 것

모듈 기능 전환 방식
워크플로우 엔진 케이스 진행 상태 관리, 단계별 처리 흐름 그대로 재활용. 케이스 유형만 교체.
SLA 타이머 처리 기한 자동 계산 및 알림 발송 소멸시효·민원 기한 관리에 그대로 적용.
배정 대시보드 담당자 배정, 진행 현황 모니터링 변호사 검토 배정에 그대로 사용.
고객 포털 기관(행원) 로그인, 권한 관리, 이력 조회 그대로 재활용. UI 텍스트만 교체.

새로 만들 것

모듈 내용 개발 기간
서류 템플릿 엔진 10가지 케이스 × 평균 3.4개 = 34종 서류 템플릿 설계 및 구현 4~6주
AI 초안 생성 입력 정보 → 서류 초안 자동 생성 (Claude API 기반) 2~3주
RAG 교체 금융법·채권법·금감원 규정 기반 RAG 재구성 2~3주
케이스 선택 UI 행원 친화적 케이스 선택 화면 및 체크리스트 1~2주
PDF 출력 법원·금감원 제출 양식에 맞는 PDF 생성 1주
총 개발 기간
8~12주 (2~3개월). 아파트집사 엔진을 재활용하므로 처음부터 만드는 것 대비 개발 기간 50% 단축. 핵심 차별화 기능(서류 템플릿·AI 초안·금융 RAG)에만 집중 투자.

차별점 01

금융권 완전 특화

경매, 소멸시효, 금감원 민원, NPL — 금융기관 고유 케이스를 모두 커버하는 유일한 플랫폼.

차별점 02

행원 B2B 전용 설계

개인 고객 대상(B2C)이 아닌 금융기관 행원을 위한 워크플로우. 법무팀 없는 소형 금융기관의 빈자리를 채운다.

차별점 03

서류 + 법무법인 직결

서류 자동화에서 멈추지 않는다. 필요 시 법무법인 황해와 직접 연결되는 신뢰 보증 구조.

차별점 04

레퍼런스 축적

기관의 법무 노하우가 플랫폼에 쌓인다. 담당자 교체에도 이력이 유지되는 기관 자산화.
